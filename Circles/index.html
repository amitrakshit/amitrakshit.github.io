Solution

• ∠BAN and ∠NMB are the angles subtended by the same segment BNMA.
• ∴ ∠BAN = ∠NMB = 30° [by theorem].
• Also, AB is the diameter. Hence,
  ∠ANB = 90°.
• In △ABN, ∠ANB = 90° and ∠BAN = 30°.
• ∴ ∠ANB + ∠BAN + x = 180°
   ⇒        90° + 30° + x = 180°
   ⇒                           x = 60°
